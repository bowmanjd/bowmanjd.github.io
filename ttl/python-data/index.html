Should I use Python 2 or 3?

Use Python 3.

Anything else is just prehistoric nonsense.

Python 2.7 will not be maintained after this year.

Installing Python 3

  • Mac: brew install python
  • Windows: scoop install python or choco install python
  • Linux: apt install python3 or dnf install python3
  • Web: repl.it/languages/python3 or python.org/shell
  • If you avoid package managers: python.org/downloads/

def greet(greeting="Hello", audience="World"):
    return f"{greeting}, {audience}!"
